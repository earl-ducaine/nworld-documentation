About this Document

This guide describes how to generate animation for use with the Sony PlayStation, using N-Geometry, N-Dynamics, the Vector Editor and the Action Editor.

Contents of this Document

The information in this manual is presented as follows:

  • Chapter 1, "Using PlayStation Express" provides an overview of the PlayStation Express process. Look here for a description of MIME objects and MIME animation, as well as for detailed background on the way PlayStation Express works.

  • Chapter 2, "Objects and TOD Animations" is a tutorial on creating TOD animations. This chapter also shows you how to preview attributed objects on the Sony PlayStation.

  • Chapter 3, "MIME: Using Displacements" is a basic tutorial on creating a MIME animation based on displacements saved on an object in N-Geometry.

  • Chapter 4, "MIME: Using Motion Capture Data" shows you how to create and animate a skeleton using motion capture data, then use the Vector and Action Editors to create an animation for the Sony PlayStation.

  • Chapter 5, "The Vector Editor" includes an overview of the Vector Editor, and describes the commands under each menu.

  • Chapter 6, "The Action Editor" provides an overview of the Action Editor, and describes the commands under each menu.

  • Chapter 7, "Sony PSX Materials and Attributes" provides an overview of materials and attributes, as well as a complete description of the attributes available in N·World's SonyPSX attribute domain.
