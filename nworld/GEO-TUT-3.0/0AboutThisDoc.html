About This Document

This document describes how to create, edit, and model using N-Geometry.

Contents of this Document

  • Chapter 1, "N-Geometry Basics"describes some basic concepts required to understand N-Geometry.

  • Chapter 2, "Constructing an Airplane" shows how to build a basic inorganic object (in this case an airplane). You'll learn the following commands: Extrude, Move along Axis, Scale, Rotate, Undo, and Chip Off. Also learn to Restructure and Rename objects. Saving objects

  • Chapter 3, "Building a Mansion" describes some additional commonly used N-Geometry commands, including: Scale XYZ, Flatten, Cut, Mirror, Plane Cut>Truncate, Axis Move, and Extrude Region.

  • Chapter 4, "Smooth with History" shows how to use the Smooth with History operation to create smoothed copies of an object, using the original as a control object. Then learn how to substitute the hi-res versions for the lo-res in an N-Dynamics script.

  • Chapter 5, "Magnet Operations" describes how to use N-Geometry's magnet technique to distribute the effect of a command over a selected area of a model.

  • Chapter 6, "Dungeon Environment" describes how to use N-Geometry's extrude and bridge functions to create a dungeon environment complete with interconnected hallways.

  • Chapter 7, "Hand" describes some techniques for building organic models.

  • Chapter 8, "Text" explains how to add text to a scene, convert text from 2D to 3D objects, deholing, cutting faces, and a brief description of freezing objects.

  • Chapter 9 "Cathedral" describes how to use the sketch editor to create detailed wires, create laminae, mirror objects, merge regions, and touches briefly on multiple extrude.

  • Chapter 10, "Elements and Parts" describes how to use some of N-Geometry's tools to select elements on a body and how to use face and vertex parts.

  • Chapter 11, "Adding Geometry" describes some basic techniques on adding detail to selected areas of an object, including a discussion of the commands Cut, Rdge Subdivide, Vertex Subdivide, and Smooth.

  • Chapter 12, "Removing Geometry" some methods for reducing polygon counts on an object, including a discussion of the commands Dissolve, Collapse, Merge Region, and Derez.

  • Chapter 13, "Displacements & Gestures" teaches you how to create displacements, which are different, saved states of a given object. It also describes how to combine displacements into gestures.

  • Chapter 14, "Fantasy Tree" describes how to create a tree for a fantasy setting. Sketch a bezier wire, change the sampling of a wire, and collapse nodes to change the topology of a region.

  • Chapter 15, "Stadium" describes how to build another kind of environment, in this case a sports stadium, using a contour solid.

  • Chapter 16, "Tubes" describes a simple technique for building another relatively low-polygon count environment in only a few steps using tubes.

  • Chapter 17, "Reinstance, Copy, & Multiply" describes the difference between three techniques that can be used to "duplicate" an object in N-Geometry: Copy, Reinstance, and Multiply. You'll also learn how to create various types of lattices using the Multiply command.

  • Chapter 18, "Mappers" describes mappers; the different types, how to create and attach them, and how to use them.

  • Chapter 19, "Unmodeling" describes some advanced modeling techniques using some of N-Geometry's unique features.

  • Chapter 20, "Fixing objects" describes the difference between three techniques that can be used to "duplicate" an object in N-Geometry: Copy, Reinstance, and Multiply. You'll also learn how to create various types of lattices using the Multiply command.

Document Style Conventions

This document contains a number of style conventions designed to make it easier to use. Different typefaces indicate such things as screen messages and operator entries. The following sections describe the style conventions used.

Notes

Notes of special interest are set off from the text within horizontal rules:

Note. Notes provide information that is important enough to demand special attention. The information may be relevant to the current discussion but awkward to include without disrupting the flow of information.

Mouse and Keyboard Notation

The following conventions are used in the text:

Table 1 Style conventions
Style Description
(SHIFT-L)

 Key combinations are enclosed in parentheses. (SHIFT-L) means to click the left mouse button while holding down the SHIFT key.

(CTRL-L) (ALT-L)

 Special modifier keys are also enclosed in parentheses.

(SHIFT-CTRL-L)

 Combinations of modifier keys are also possible.

Menu Notation

Menu items that are preceded and followed by a period have multiple selections, which you select by clicking left, middle, or right on the mouse. (The different options are shown on the mouse documentation line, displayed at the bottom of the screen.)

Mouse Documentation Line Notation

Due to space constraints, instructions on the mouse documentations line (across the bottom of the monitor) are slightly different. The following conventions are used on the mouse documentation line:

Table 1 Style conventions
Style Description
L, M, R

 Click the left, middle, or right mouse button.

sh-L, sh-M, sh-R

 While holding down the SHIFT key, click on the left, middle, or right mouse button.

c-L, c-M, c-R

 While holding down the CTRL key, click on the left, middle, or right mouse button.

a-L, a-M, a-R

 While holding down the ALT key, click on the left, middle, or right mouse button.

sh-*, c-*, a-*

 Click any mouse button while holding down the appropriate modifier key (SHIFT, CTRL, or ALT)

+sh, +c, +a

 In addtion to the specified key combination, hold down the CTRL, ALT, or SHIFT key. You may, for example, need to hold down both the ALT and SHIFT keys before you click the appropriate mouse button.
