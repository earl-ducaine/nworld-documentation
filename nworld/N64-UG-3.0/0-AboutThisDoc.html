About this Document

This guide describes Nichimen's N64 Express, which lets you interactively preview objects created in N·World on the Nintendo 64, then export them in Nintendo display list format for use in your game.

Contents of this Document

The information in this manual is presented as follows:

  • Chapter 1 "Using N64 Express" describes how to create an object, assign N64 attributes to it, then preview it on the Nintendo 64. It also describes how to prepare and preview animations on the N64.

  • Chapter 2 "N64 Materials and Attributes" describes the N64 attribute domain, designed to allow you to create detailed texture maps for use on the N64.
