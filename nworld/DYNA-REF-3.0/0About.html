About this Document

This guide describes how to use the N-Dynamics animation choreographer. It describes the basic concepts in N-Dynamics and describes what each of the operations does.

Contents of this Document

The information in this document is presented as follows:

  • Chapter 1, "Overview" gives a basic description of how N-Dynamics works.

  • Chapter 2, "The Interface"describes the basic elements of the N-Dynamics interface.

  • Chapter 3, "Using the Script Editor"describes how to use the script editor to create, edit, and animate scripts.

  • Chapter 4, "Working with Curves"describes how curves are used by N-Dynamics. It describes the various types of curves available and how to use the curve editor.

  • Chapter 5, "Dynamic Operations"describes each of the dynamic operations that can be animated in N-Dynamics.

  • Appendix A, "Hot Keys" describes the hot key shortcuts available in N-Dynamics.
