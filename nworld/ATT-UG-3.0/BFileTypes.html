File Types

File Types Created by N-Render

N-Render creates the types of files listed in Table B.1:

Table B.1 Render file types
Type Description
.pg

 Polygon data file.

.vx

 Vertex data file.

.ltg

 Light group file. An editable text file which lists the various lights associated with each light group.

.lat

 Light attributes file. An editable text file, showing the attributes associated with each light in the scene.

.sat

 Surface attributes file. An editable text file that describes the object attributes.

.ast

 Animation script file.

.tpx
.tpx.Z
.tif
.sgi
.rgb

 The image file.

Note. tpx.Z is a compressed tpx file.

While it is possible to edit the text files describing an object or a light using a text editor, it is generally easier to do so through the N-Render attributes interface. You should modify the files themselves only if instructed to do so by Nichimen Graphics Support.
