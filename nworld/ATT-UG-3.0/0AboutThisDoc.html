About this Document

This document describes how to create and edit both render materials and lights using the Attributes Editor. This document describes

Contents of this Document

  • Chapter 1, "Rendering Concepts" describes basic rendering concepts, and provides an overview of the N·World rendering process. This chapter introduces the Attributes Editor window and some of its more basic features.

  • Chapter 2, "Creating & Using Materials" describes how to define materials and apply them to objects.

  • Chapter 3, "Render Domains" describes how to define the default render parameters for each render domain, and the attributes that can be specified in both the GL Shade and Render domains. This chapter also includes an overview of image mapping techniques used with N-Render.

  • Chapter 4, "Lights" describes how to create and modify lights in the N-Geometry environment, and how to use the Light Group Editor to assign light groups to diffferent materials.

  • Appendix A, "TPX File Format" describes the structure of the TPX file format (N-Render's native file format).

  • Appendix B, "File Types" describes the various types of files written out by N-Render and their extensions.

  • Appendix C, "Increasing Render Speed" offers several tips for increasing the speed of renders using N-Render, and described which features are more time intensive.
