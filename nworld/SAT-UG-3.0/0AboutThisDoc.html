About this Document

This guide describes the Saturn Express, which lets you interactively preview objects created in N·World on the Sega Saturn, then export them in SGL format for use in your game.

Contents of this Document

 The information in this manual is presented as follows:

  • Chapter 1, "Installing the Sega Cartdev" describes how to install the Sega Cartdev, and also how to install the Sega compiler.

  • Chapter 2,"Using Saturn Express"describes how to preview and export objects and textures on the Sega Saturn. It also describes how to export objects and textures to SGL format, and how to compile these files for use in games

  • Chapter 3, "Sega Materials and Attributes" contains a brief overview of N·World materials, as well as parameters and attributes for the Sega domain.

  • Chapter 4, "Functions" describes some of the functions available in the Sega API.
