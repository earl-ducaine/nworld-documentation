About this Document

This guide describes how to use the Skeletal Animation System (SAS). It includes tutorials in building and animating skeleton-based animation.

Contents of this Document

  • Chapter 1, "Getting Started"describes some basic concepts required to understand the Skeletal Animation System, and creates a basic script to animate a skeleton using motion capture data.

  • Chapter 2, "Building Skeletons" shows some of the tools that can be used to build both humanoid and non-humanoid skeletons.

  • Chapter 3, "Attaching Objects"shows different methods for attaching terminal objects, object hierarchies, or a set of default objects to skeletons, bones, or collections of bones.

  • Chapter 4, "Skinning a Skeleton"describes how to assign a skin to a skeleton, how to edit skin parts, and how to creat your own skin parts in order to perform "auto-skinning."

  • Chapter 5, "Skinning Enhancements" describes techniques for enhancing the behavior of skins on a skinned skeleton; using soft parts, creating skin displacements, and using extra bones to drive secondary skin displacements.

  • Chapter 6, "Posing a Skeleton" shows a variety of different methods for posing skeletons.

  • Chapter 7, "Making a Walk Cycle" describes how to create a simple pose-to-pose animation and how to create motion cycles.

  • Chapter 8, "Walk Cycle Enhancements" offers tips for improving a typical animation, including using generated "wires" as guides for animated skeletons and layering artist motion over motion capture data.

  • Chapter 9, "Cartoon Effects" shows how to animate the lengthening of a bone to create cartoonish effects for a character.

  • Chapter 10, "Editing Motion Capture Data" describes modifying motion capture data, including methods for layering artist specified motion, decomposing motion capture data into rotational components or poses, and blending poses into motion capture. It also describes how to use the Motion Transform command to use the same motion data on skeletons of different sizes and proportions.

  • Chapter 11, "Motion Blending" describes how to blend between motion capture sequences (e.g., from a walk to a run).
