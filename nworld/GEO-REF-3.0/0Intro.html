About This Document

This guide describes how to use the N-Geometry modeler. It describes the basic concepts in N-Geometry and describes what each of the operations does.

Contents of this Document

The information in this document is presented as follow:

  • Chapter 1, "Overview"describes some basic concepts about N-Geometry. It describes how to control the camera, parts of the N-Geometry interface, and how to perform many commonly used tasks.

  • Chapter 2, "File and Edit Menus"describes how to add new objects in N-Geometry, how to load and save objects, how to delete objects, and how to export objects to different file formats.

  • Chapter 3 "Working with Objects" describes how to work with objects in N-Geometry. Commands that affect an entire object are called transformations. You work with entire objects when you're setting up an animation.

  • Chapter 4 "Working with Bodies" describes how to work with bodies in N-Geometry. Commands that affect only the body of an object (or its elements) are called modifcations. You work with a body when you're sculpting the object.

  • Chapter 5 "Viewing Objects" describes the various parameters that can be set for your view window-toggling the display of visible axes, saving camera views, and other camera related commands.

  • Chapter 6 "Object Display" describes the various display parameters that can be set when viewing objects-how to make objects visible/invisible, shaded, and so forth.

  • Chapter 7 "Utilities" describes utility-oriented commands available in N-Geometry. Setting global preferences, restructuring objects, renaming, displaying information about objects, and creating custom menu configurations are all described in this chapter.

Menu items that are preceded and followed by a period have multiple selections, which you select by clicking left, middle, or right on the mouse. (The different options are shown on the mouse documentation line, displayed at the bottom of the screen.)
