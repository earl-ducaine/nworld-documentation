About this Document

This reference guide describes the various Skeletal Animation System commands found in both N-Geometry (where you and N-Dynamics.

Contents of this Document

  • Chapter 1, "Glossary of Terms" defines a number of terms commonly used in working with skeletons.

  • Chapter 2, "Skeletons in N-Geometry" describes the commands available in N-Geometry for creating and modifying skeletons and bones. This is intended as a supplement to the N-Geometry Reference Guide.

  • Chapter 3, "Dynamic Operations" describes how to use the various dynamic operations in N-Dynamics that work with skeletons. This is intended as a supplement to the N-Dynamics Reference Guide.
